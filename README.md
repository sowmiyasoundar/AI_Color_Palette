## AI Color Palette Generator | React + Vite

Live Demo: https://ai-color-palette-rose.vercel.app/

AI Color Palette Generator is a web-based application that generates theme-based color palettes using AI.
Users can enter any theme (for example: forest, sunset, pastel summer) and receive 10 pure HEX color codes generated by an AI model.
The interface is minimal, responsive, and includes a dark mode switch with a professional layout.

## Features

Generate 10 AI-powered HEX color codes

Clean and responsive UI

Dark/Light mode toggle

One-click color copying

Deployed via Vercel

## Tech Stack

Frontend: React + Vite

Styling: Tailwind CSS

API: OpenRouter (LLaMA-3 model)

HTTP Client: Axios

Deployment: Vercel

## API Setup (OpenRouter)

Create a .env file in the project root and add:

VITE_OPENROUTER_KEY=your_api_key_here

## Installation

Clone the repository

git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name


## Install dependencies

npm install


## Start development server

npm run dev


## Build for production

npm run build

## Deployment (Vercel)

Push your project to GitHub

Visit https://vercel.com

Import the repository

Add your environment variable:

VITE_OPENROUTER_KEY


Deploy

## Usage

Enter a theme such as neon glow, minimal luxury, pastel beach, etc.

Click Generate Palette

The app will return 10 HEX colors

Click any color to copy its code

Switch between dark or light mode anytime
